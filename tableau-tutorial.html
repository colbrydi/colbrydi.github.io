Tableau Tutorial

Wed 04 May 2022 by Carlie Couzens

Written by Carlie Couzens

As part of my honors project for CMSE495, I created a Tableau walkthrough which includes information on how to set up and get started with Tableau. Additionally, I created a full tutorial on how to load a geographical dataset and create a map with additional optional personalizations. This tutorial serves as a prime example of the Tableau workflow (clean data inputted → graphical configuration → personalization). While it is possible to clean data within the Tableau platform and even create new calculated fields, this process is streamlined by using Python beforehand to clean data. My hope is this tutorial will provide a launching point for future students who are interested in Tableau but might not know where to begin.
